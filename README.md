**Terrixa** is a premium, high-performance voxel game-creation platform. Explore infinite worlds, build incredible structures, and create your own adventures with our advanced modding system.

---

## 🎮 How to Play

### **1. Download**
Download the latest release from our [GitHub Releases](https://github.com/snikdev/terrixa/releases) or the main repository.

### **2. Installation**
1. Extract the downloaded ZIP file to a folder of your choice (e.g., `C:\Games\Terrixa`).
2. Open the folder and locate the **`TerrixaLauncher.bat`** file.

### **3. Launch**
- Double-click **`TerrixaLauncher.bat`** to open the launcher.
- Click **"Play Terrixa"** to start the engine.
- *Note: Using the launcher ensures the game runs with the correct branding and window settings.*

## 🚀 Key Features
- **Stunning Visuals**: Enhanced UI and custom branding.
- **Infinite Exploration**: Procedurally generated worlds that go on forever.
- **Modular System**: Easy to add or remove games, mods, and texture packs.
- **Developer Friendly**: Built on the flexible Luanti engine with full Lua support.
