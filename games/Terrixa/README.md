# Terrixa

Terrixa is a simple and peaceful sandbox game, with no
real goals or built-in enemy mobs.

While it provides a basic gameplay experience out of the box, it is best played
with mods.

Terrixa can be extended with your own mods and content packs.

## Installation

### In-App Browser

* Content > Browse Online Content
* Search for Terrixa packages
* Click Install

### Manually

- Unzip the archive, rename the folder to `Terrixa` and
place it in `.../minetest/games/`

- GNU/Linux: If you use a system-wide installation place it in `~/.minetest/games/`.

The Luanti engine can be found at [GitHub](https://github.com/minetest/minetest).

For further information or help, see: [Installing Mods](https://wiki.luanti.org/Installing_Mods).

## Compatibility

As of January 2024 Terrixa follows a **rolling release** model with new changes being automatically
published on ContentDB every day.

There are no stable releases or tags, all existing ones are considered legacy.

Terrixa is always compatible to the latest stable release of the engine.
Older releases may be supported too depending on circumstances.
